# Rule1

Rule1 is a gem that locates Rule 1 trading opportunities.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rule1'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rule1
